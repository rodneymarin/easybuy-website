# Información y Referencias del Proyecto: EasyBuy

## Descripción del Proyecto
Website de producto (Product Page) para **EasyBuy**, una aplicación móvil para Android desarrollada en **React Native + Expo**, que también cuenta con su respectiva versión Web. Su función principal es crear y gestionar listas de compras.

---

## Funcionalidades Principales
* **Crear múltiples listas de compras:** Permite organizar diferentes listas independientes.
* **Maestro de productos:** Gestión y catálogo de productos.
* **Maestro de tiendas:** Gestión de establecimientos y comercios.
* **Precios por tienda:** Permite registrar y asociar precios específicos para cada producto según la tienda.
* **Gestión de ítems:** Agregar productos directamente a las listas.
* **Asignación de tienda:** Permite indicar en qué tienda específica se comprará cada producto de una lista.
* **Seguimiento de compras:** Permite marcar productos como completados.
* **Acciones avanzadas (Long Press):** Permite marcar productos de la lista mediante pulsación prolongada para desplegar opciones para moverlos a otra lista o eliminarlos.
* **Exportación de datos:** Permite exportar la lista al portapapeles en formato texto.
* **Modos de almacenamiento:** Permite trabajar de forma local (base de datos local) o conectado a la nube creando una cuenta remota (ambas bases de datos trabajan con datos separados).
* **Multiplataforma:** Se puede instalar el APK en Android o acceder directamente a la versión Web online.
* **Personalización e Idioma:** Es multi-idioma (Español / Inglés) y soporta tema claro y oscuro.
