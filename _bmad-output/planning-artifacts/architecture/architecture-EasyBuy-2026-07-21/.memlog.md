- (decision) AD-1: Stack — Vite + vanilla HTML/CSS/JS. Static landing page no necesita framework JS; Vite da build rápido, recarga en caliente, manejo de assets.
- (decision) AD-2: CSS — Tailwind CSS. Utilidades directas, consistente con color primary #4a5df9, responsive sin media queries manuales.
- (decision) AD-3: Assets — SVG como archivos .svg en /assets/, nunca inline. Permite reemplazar diseños sin reescribir código.
- (decision) AD-4: Secciones modulares — Cada sección del sitio en su propio archivo HTML partial importado vía Vite. Facilita edición independiente sin tocar el layout completo.
- (decision) AD-5: QR code — Generado estáticamente como archivo .svg en /assets/, no runtime.
- (decision) AD-6: SEO — Meta tags básicos en <head>: title, description, Open Graph básico. Sin analytics ni tracking.
- (decision) AD-7: Deploy — Salida dist/ estática. Compatible con cualquier host (Vercel, Netlify, GitHub Pages, servidor simple).
